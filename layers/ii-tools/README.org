* Description
This layer adds a set of useful all-around functions for our work.
** Features:
  - find your ssh agent to enable pushing to git with ssh
  - a timesheet templating tool, following stephen's timesheet flow
* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =ii-tools= to the existing =dotspacemacs-configuration-layers= list in this
file.
